# Friendzone
The friend zone app

# Backend
# To start docker
move into `src` then `docker`

ensure your docker desktop is running.
run `docker compose build` 
run `docker compose up -d`

# Frontend
### Starting up
Add packages by running `yarn`, then run the Front end server with `yarn dev`
